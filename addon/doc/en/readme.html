Battery Check

The purpose of this add-on is to monitor the user's battery so that, when the established limit is reached (by default 100%), an alert begins to be issued every 10 seconds and thus prevents battery wear.

How to use it?

By default there is no pre-assigned gesture, so you have to go to the NVDA menu>Preferences>Input gestures>Battery Check>Start or stop battery monitoring and assign one.
Once established, just run it and if monitoring is activated it will be deactivated, and if it is not, the opposite will happen.
